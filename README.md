Tr8n Struts 2 Sample Application
==================
[![Project status](http://stillmaintained.com/tr8n/tr8n_samples_welp_struts2.png)](http://stillmaintained.com/tr8n/tr8n_samples_welp_struts2.png)

This sample application demonstrates how to use Tr8n in Struts 2 environment.

Installation
==================

If you are using Maven, use the following procedures:

```bash
$ git clone https://github.com/tr8n/tr8n_samples_welp_struts2.git
```

Import the project into Eclipse using the Maven plugin and run the web application.
